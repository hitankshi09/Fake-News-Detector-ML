# Fake-News-Detector-ML
Fake News Detector is a Machine Learning project that classifies news articles as fake or real using Natural Language Processing. It uses TF-IDF and Logistic Regression for prediction and provides a simple web interface built with Streamlit.
A Machine Learning project that detects whether a news article is Fake or Real using Natural Language Processing. The model analyzes text patterns using TF-IDF and predicts authenticity with Logistic Regression.

The model was trained on political news articles, so it will perform best on topics like:
Government
Elections
International relations
Policies
Political scandals

** Features**

News text preprocessing and cleaning
Feature extraction using TF-IDF
Machine learning classification model
Interactive prediction interface
Simple and beginner-friendly ML project
